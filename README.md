Project Title: College Management System

Summary:
Developed a comprehensive College Management System using Python (Tkinter) and MySQL to streamline student and teacher record management. The system improves administrative efficiency and offers secure, cloud-based data access. Built using the Agile methodology for flexible and progressive development.

Key Responsibilities & Achievements:

Designed and developed both front-end (Tkinter GUI) and back-end (MySQL) components.

Created a user-friendly interface for easy navigation and data entry.

Implemented modules for student profiles, subject registration, grades, and payment history.

Managed teacher and student records, scheduling, and curriculum tracking.

Ensured data security and remote accessibility through cloud integration.

(If applicable) Led the team as Project Manager, tracking progress and ensuring timely delivery.

(If applicable) Handled programming and database management as the project’s main developer.

Skills Used:
Python · Tkinter · MySQL · Agile Development · CRUD Operations · Software Development · Database Management · Project Management
