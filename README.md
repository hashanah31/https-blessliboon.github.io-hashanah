Project Title: Queue System for Addressing Late Arrivals in Jeepney Transportation

Summary:
Developed a Java-based queue system to address late arrivals in jeepney transportation in the Philippines. The project aimed to improve passenger convenience, operational efficiency, and sustainability. Designed following systematic research and implementation methods from Chapters 1–3, covering problem identification, literature review, system design, and evaluation.

Key Responsibilities & Achievements:

Identified key issues causing jeepney delays and proposed a queue-based reservation solution.

Developed a functional Java system using a menu-driven interface for seat reservation, queue checking, and ticket printing.

Implemented queue management through LinkedList to organize passenger flow efficiently.

Addressed scope and limitations, including geographical area, operational challenges, and data reliability.

Delivered a research-backed system promoting better passenger experience and reduced congestion.

Java Programming · Queue Management · System Design · Problem Solving · Research & Data Analysis · Policy Analysis · Technical Writing
